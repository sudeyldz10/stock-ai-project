# Stock Price Predictor

A simple terminal-based stock analysis tool built with Python.

This project was created to practice Python programming fundamentals, working with financial data, machine learning basics, and data visualization.

## Features

- Fetch real-time stock data via Yahoo Finance
- Predict next price using a Linear Regression model
- Get a simple BUY / SELL signal based on the prediction
- Visualize price history and prediction on a graph

## Technologies Used

- Python
- yfinance
- scikit-learn
- matplotlib
- numpy
- Lists & Dictionaries
- Functions
- Modules
